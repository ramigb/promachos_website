üë• Team Workflows

Handoff Process

  1. Developer A: Works with AI, protocol auto-updates
  2. Developer B: Runs promachos start
  3. AI Assistant: Receives full context automatically
  4. Seamless Continuation: No information loss

Best Practices

  • Always run promachos start when taking over work
  • Let AI update protocol files after significant changes
  • Use promachos status for team standups
  • Maintain consistent collaboration standards

üîç Troubleshooting

Common Issues

Protocol not detected by AI

Ensure the AI instruction file (e.g., CURSOR.md) exists in your project root

Context not updating

Check that config.yaml has update_progress: true and track_decisions: true

Installation issues

Try npm install -g promachos --force or check Node.js version compatibility
